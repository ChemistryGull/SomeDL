# SomeDL WebUI
This is a quick image collection of the SomeDL WebUI.

## Download Section
Download songs by Youtube URL and search queries. Check the download status of each song. 
Use the pause control in the bottom left to pause the download and use the trash icon to remove queued downloads.
Click on the question mark for helpful information.
![SomeDL WebUI](https://github.com/ChemistryGull/SomeDL/blob/main/docs/images/webui/somedl_webui_download.png)
