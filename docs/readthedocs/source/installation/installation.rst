Installation
============

Windows
-------

Install via pip:

.. code-block:: bash

   pip install somedl

or

.. code-block:: bash

   python -m pip install somedl

To update an existing SomeDL installation, add the ``--upgrade`` flag tho the command above.

Linux
-----
This software is currently not packaged in any repository. 
Install it using your preferred Python package manager, such as `pipx <https://github.com/pypa/pipx>`_:

.. code-block:: bash

   pipx install somedl
